# 本家AI - 百度AI服务配置指南

## 🚀 概述

本家AI已集成百度AI开放平台的照片修复服务，包括：
- **智能修复**：修复划痕、污渍等损伤
- **照片上色**：为黑白照片智能上色
- **清晰增强**：提升分辨率和清晰度
- **降噪处理**：去除噪点和颗粒感

## 📋 配置步骤

### 1. 注册百度AI开放平台

1. 访问 [百度AI开放平台](https://ai.baidu.com/)
2. 注册百度账号并登录
3. 进入控制台，创建应用

### 2. 开通图像处理服务

1. 在控制台中选择"图像处理"
2. 开通以下服务：
   - 图像修复
   - 图像增强
   - 图像去噪
   - 黑白图像上色

### 3. 获取API密钥

1. 在应用列表中找到您的应用
2. 查看并复制 `API Key` 和 `Secret Key`

### 4. 配置环境变量

在项目根目录创建 `.env.local` 文件：

```bash
# 百度AI API配置
NEXT_PUBLIC_BAIDU_AI_API_KEY=your_api_key_here
BAIDU_AI_SECRET_KEY=your_secret_key_here
```

### 5. 启用真实API

编辑 `src/lib/ai-apis/baidu-ai.ts` 文件：

```typescript
// 将此行修改为 false 启用真实API
const USE_MOCK_API = false;
```

### 6. 重启服务

```bash
npm run dev
# 或
bun dev
```

### 使用流程

1. **登录系统**：注册并登录本家AI
2. **进入AI服务**：在家族主页点击"AI服务"选项卡
3. **选择照片修复**：点击"AI照片修复"卡片

5. **选择修复类型**：根据需要选择修复方式
6. **开始处理**：点击"开始修复"按钮
7. **查看结果**：等待处理完成，对比效果
8. **下载保存**：下载修复后的照片

## 🛡️ 安全说明

### 数据安全

- 所有照片仅用于AI处理，不会被存储或用于其他目的
- API调用通过HTTPS加密传输
- 处理完成后本地临时文件自动清理

### API密钥保护

- `NEXT_PUBLIC_BAIDU_AI_API_KEY`：仅用于客户端token获取
- `BAIDU_AI_SECRET_KEY`：敏感信息，仅在服务端使用
- 建议定期轮换API密钥

## 🐛 故障排除

### 常见问题

**Q: API调用失败，返回权限错误**
A: 检查API密钥是否正确配置，确认服务是否已开通

**Q: 处理速度较慢**
A: 百度AI服务器负载较高时可能出现，请耐心等待

**Q: 某些图片处理失败**
A: 图片格式、大小或内容可能不符合API要求，请尝试其他图片

**Q: 免费额度用完了怎么办？**
A: 可以升级为付费服务，或等待下月免费额度重置

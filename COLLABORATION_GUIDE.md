# 本家AI - 协作创作功能指南

## 🤝 功能概述

本家AI现已支持**多人协作创作**功能，让整个家族的成员可以共同参与AI家族故事的创作过程。通过协作功能，家族成员可以一起分享回忆、提供信息、参与讨论，共同创作出更加丰富和真实的家族传奇故事。

### ✨ 核心特性

#### 🎭 **多角色协作系统**
- **所有者**：项目创建者，拥有全部权限
- **编辑者**：可以编辑内容、生成故事、邀请其他成员
- **评审者**：可以查看内容、添加评论和建议
- **查看者**：只能查看项目内容

#### 💬 **实时协作交流**
- 多人同时参与AI对话
- 实时查看其他成员的在线状态
- 协作活动记录和通知
- 评论和建议系统

#### 📸 **共享资源管理**
- 多人上传家族照片
- 记录每张照片的上传者
- 共享照片分析结果
- 协作编辑故事内容

#### 🔒 **权限安全管理**
- 基于角色的权限控制
- 安全的邀请机制
- 项目访问权限管理
- 操作记录和审计

## 🚀 快速开始

### 1. 创建协作项目

1. **登录本家AI系统**
   - 进入"AI服务" → "AI家族故事"
   - 点击"创建新故事"按钮

2. **配置故事设置**
   - 选择故事风格、写作语调和篇幅
   - 默认启用协作功能
   - 确认创建项目

3. **自动获得所有者权限**
   - 项目创建者自动成为所有者
   - 拥有邀请成员、管理权限等全部功能

### 2. 邀请家族成员

1. **进入协作面板**
   - 在故事创作界面点击"协作"按钮
   - 切换到"协作者"标签页

2. **发送邀请**
   - 点击"邀请成员"按钮
   - 选择要邀请的家族成员
   - 设置协作角色（编辑者/评审者/查看者）
   - 添加邀请留言（可选）
   - 发送邀请

3. **管理邀请**
   - 查看邀请状态
   - 跟踪邀请回应
   - 重新发送或撤销邀请

## 🎯 最佳实践

### 协作流程建议

#### 1. **项目启动阶段**
```
1. 项目创建者设定故事主题和风格
2. 邀请核心家族成员作为编辑者
3. 邀请其他成员作为评审者或查看者
4. 在项目中发布协作指导和分工说明
```

#### 2. **信息收集阶段**
```
1. 各成员分工收集不同时期的家族信息
2. 上传各自保存的珍贵照片
3. 分享不同角度的家族故事和回忆
4. 通过评论系统讨论和确认信息的准确性
```

### 权限分配建议

#### 👑 **所有者（1人）**
- 项目发起人或家族长辈
- 负责整体规划和协调
- 管理所有协作者权限

#### ✏️ **编辑者（2-3人）**
- 熟悉家族历史的核心成员
- 负责主要的内容创作和AI对话
- 收集和整理家族信息

#### 📝 **评审者（3-5人）**
- 不同年龄段的家族成员
- 提供多角度的信息和建议
- 确保故事内容的准确性

#### 👀 **查看者（不限）**
- 其他家族成员
- 关注项目进展
- 参与最终成果的分享

# 本家AI - AI纪念空间功能指南

## 🕯️ 功能概述

AI纪念空间是本家AI的温馨功能，为逝去的家族成员创建虚拟家祠和纪念场所。通过AI技术和游戏化互动，让家族成员可以以更加温馨、有意义的方式缅怀和纪念逝去的亲人。

### ✨ 核心特性

#### 🏛️ **虚拟家祠设计**
- **四种主题风格**：花园式、烛光式、现代简约、传统中式
- **3D立体场景**：沉浸式的纪念空间体验
- **个性化装饰**：根据逝者特点定制纪念元素
- **动态背景**：优美的动画效果和音效

#### 🤖 **AI设计助手**
- **智能对话**：与AI讨论家祠设计和装饰方案
- **设计建议**：AI根据逝者特点提供个性化建议
- **实时优化**：通过对话不断完善纪念空间
- **文化融合**：结合传统文化和现代审美

#### 🌺 **游戏化献花系统**
- **6种花卉类型**：玫瑰、百合、菊花、康乃馨、向日葵、郁金香
- **积分机制**：通过参与获得积分用于献花
- **寓意表达**：每种花卉都有特殊的纪念寓意
- **实时装饰**：献花会实时显示在虚拟家祠中

#### 🎵 **背景音乐系统**
- **宁静音乐**：播放安详的纪念音乐
- **音量控制**：可调节背景音乐音量
- **沉浸体验**：音乐与视觉效果完美结合
- **个性选择**：可选择逝者生前喜爱的音乐

## 🚀 使用流程

### 1. 创建纪念空间

#### 访问功能
- 登录本家AI系统
- 进入"家族主页" → "AI服务"
- 点击"AI纪念空间"卡片

#### 选择主题
```
🌸 花园式主题
   - 温馨花园场景，充满生机与希望
   - 适合性格开朗、热爱生活的逝者

🕯️ 烛光式主题
   - 温暖烛光氛围，寄托无尽思念
   - 适合温柔慈祥、充满爱心的逝者

⭐ 现代简约主题
   - 简洁优雅设计，宁静致远
   - 适合知识分子、追求简约的逝者

👑 传统中式主题
   - 古典庄重风格，传承文化
   - 适合传统观念、德高望重的逝者
```

#### 上传纪念照片
- 支持JPG、PNG格式
- 建议上传清晰的生活照片
- 最多可上传10张照片
- AI会分析照片特征进行个性化设计

### 2. AI对话优化设计

#### 与AI设计助手交流
```
用户: "我想为爷爷设计一个庄重的纪念空间"
AI: "我建议选择传统中式主题，在家祠中央放置爷爷的照片，
    两侧添加青铜香炉，背景使用温暖的金色调..."

用户: "爷爷生前最喜欢菊花"
AI: "那我们可以在家祠周围装饰菊花元素，菊花象征高洁的品格，
    很适合表达对爷爷的敬意和思念..."
```

#### AI设计建议类型
- **装饰建议**：根据逝者性格推荐装饰元素
- **颜色搭配**：建议最适合的色调方案
- **花卉选择**：推荐有纪念意义的花卉
- **音乐推荐**：建议合适的背景音乐
- **文化融合**：结合传统文化元素

### 3. 游戏化献花互动

#### 献花系统
| 花卉 | 积分 | 寓意 | 适用场景 |
|------|------|------|----------|
| 🌹 玫瑰 | 1积分 | 深深的爱 | 表达对逝者的深情 |
| 🤍 百合 | 2积分 | 纯洁的心 | 寄托纯真的思念 |
| 🌼 菊花 | 1积分 | 思念如丝 | 传统的悼念花卉 |
| 🌺 康乃馨 | 2积分 | 永恒的爱 | 母亲节、表达感恩 |
| 🌻 向日葵 | 3积分 | 阳光希望 | 积极向上的纪念 |
| 🌷 郁金香 | 2积分 | 美好祝愿 | 祝愿逝者安息 |

#### 获得积分方式
- **创建纪念空间**：+50积分
- **上传照片**：每张+5积分
- **与AI对话**：每次+2积分
- **邀请家族成员**：每人+10积分
- **分享纪念空间**：+20积分

### 4. 虚拟家祠体验

#### 沉浸式场景
- **360度视角**：可以从不同角度欣赏家祠
- **动态效果**：飘落的花瓣、闪烁的烛光等
- **实时装饰**：献花后立即显示在场景中
- **音效配合**：轻柔的背景音乐营造氛围

#### 互动功能
- **全屏模式**：获得更好的沉浸体验
- **拍照保存**：保存纪念空间的美好瞬间
- **分享功能**：与家族成员分享纪念空间
- **留言功能**：为逝者留下思念的话语

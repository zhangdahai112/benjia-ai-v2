# 本家AI - 语音交互功能指南

## 🎤 功能概述

本家AI现已全面支持语音交互功能，让您可以通过语音与AI助手自然对话，创作家族故事变得更加便捷和有趣！

### 🚀 核心功能

#### 1. **智能语音识别**
- 🗣️ **自然对话**：支持中文语音识别，准确率高
- 🎯 **实时转录**：语音实时转换为文字，过程可视化
- 🔄 **自动发送**：语音识别完成后自动发送给AI
- 📊 **置信度显示**：显示语音识别的准确度

#### 2. **语音合成播放**
- 🔊 **AI回复朗读**：可以朗读AI的回复内容
- 🎵 **自然语音**：使用浏览器原生语音引擎
- ⚙️ **语音控制**：支持播放、暂停、停止操作
- 🌍 **多语言支持**：支持中英文语音合成

#### 3. **双模式输入**
- ⌨️ **文字模式**：传统的文字输入方式
- 🎤 **语音模式**：语音输入模式，一键切换
- 🔄 **无缝切换**：随时在两种模式间切换
- 💡 **智能提示**：实时状态提示和使用建议

## 📋 使用方法

### 启用语音功能

1. **进入AI故事创作界面**
   - 登录本家AI系统
   - 进入"AI服务" → "AI家族故事"
   - 创建或打开一个故事项目

2. **切换到语音输入模式**
   - 在聊天界面底部找到输入模式切换按钮
   - 点击"语音输入"切换到语音模式
   - 界面会显示语音输入组件

### 语音输入操作

1. **开始语音输入**
   ```
   🎤 点击麦克风按钮开始录音
   📢 清晰地说出您想表达的内容
   ⏸️ 说完后停顿，系统会自动识别结束
   ✅ 识别完成后自动发送给AI
   ```

2. **语音状态指示**
   - 🔴 **录音中**：麦克风图标变红，有动画效果
   - 🔵 **处理中**：显示加载动画，AI正在处理语音
   - ✅ **完成**：显示识别结果和置信度
   - ❌ **错误**：显示错误信息和重试选项

3. **语音播放功能**
   - 在右侧语音助手面板中输入要朗读的文字
   - 点击播放按钮听取AI的回复
   - 支持播放控制：播放/暂停/停止

### 权限要求

1. **麦克风权限**
   - 首次使用会弹出权限请求
   - 请点击"允许"以启用语音功能
   - 如果拒绝，可在浏览器设置中重新开启

2. **网络要求**
   - 语音识别需要网络连接
   - 建议在稳定的网络环境下使用
   - 支持HTTPS安全连接

### 最佳实践

#### 🎯 语音输入技巧

1. **环境准备**
   - 选择安静的环境，减少背景噪音
   - 保持与设备适当的距离（30-50cm）
   - 确保麦克风工作正常

2. **说话技巧**
   - 语速适中，发音清晰
   - 避免方言或过于口语化的表达
   - 句子完整，逻辑清晰

3. **内容组织**
   - 每次语音输入控制在1-2句话
   - 复杂内容可分段输入
   - 重要信息可重复确认

## 🔧 故障排除

### 常见问题

**Q: 语音识别不准确怎么办？**
A:
- 检查麦克风是否正常工作
- 在安静环境中重试
- 说话清晰，语速适中
- 可以切换回文字输入模式

**Q: 没有声音输出怎么办？**
A:
- 检查设备音量设置
- 确认浏览器音频权限
- 尝试刷新页面重新加载

**Q: 麦克风权限被拒绝了？**
A:
- 点击地址栏的锁图标
- 重新设置麦克风权限为"允许"
- 刷新页面重新尝试

**Q: 语音功能不可用？**
A:
- 确认使用支持的浏览器（Chrome/Edge推荐）
- 检查网络连接是否稳定
- 尝试重启浏览器
